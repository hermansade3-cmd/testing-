# Sade Books

Karibu kwenye **Sade Books**, website ya kuuza vitabu vya PDF mtandaoni.

## Kuhusu Website

Website hii imeundwa kwa ajili ya kuuza vitabu vya kidigitali (PDF). 
Watumiaji wanaweza kuona vitabu, kulipia, na kupakua vitabu baada ya malipo.

## Huduma Zilizopo

- Kuonyesha vitabu
- Kuuza vitabu vya PDF
- Kupakua vitabu baada ya malipo

## Jinsi ya Kununua Kitabu

1. Fungua ukurasa wa vitabu
2. Chagua kitabu unachotaka
3. Bonyeza **Nunua Kitabu**
4. Fanya malipo
5. Utapokea link ya kupakua PDF

- GitHub Pages
